Welcome to Learn Words in Perl

Learn-words was built to make it easier to learn words. It has various ways to learn, including multiple choice questions. It doesn't count how many times you are correct, but it does tell you when you are incorrect.

Download source code at https://github.com/bean5/learn_words/zipball/master.
